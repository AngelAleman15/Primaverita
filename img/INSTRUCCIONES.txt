Imágenes de ejemplo para el viaje espacial.

Para que las imágenes aparezcan durante el viaje espacial, coloca archivos de imagen con estos nombres:

- image1.jpg / image1.png
- image2.jpg / image2.png  
- image3.jpg / image3.png
- foto1.jpg / foto1.png
- foto2.jpg / foto2.png
- space1.jpg / space1.png
- galaxy.jpg / galaxy.png
- nebula.jpg / nebula.png

El sistema buscará automáticamente estos archivos y los mostrará durante el viaje.
